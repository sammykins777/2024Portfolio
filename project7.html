Storyline Experiments

I have several years of developing learning interactions in Articulate Storyline. Here's an example of an interaction I had built for a complaince training.

This example served as a test for the cloud-based animation tool, Rive. Following Gary Simon's Design Course tutorial, I created an animated rocket, learning the ins and outs of animation, exporting, and coding for web interactivity. By embedding this animation within a Storyline web object, I experimented with the seamless transfer of variables. Clicking the button not only alters the animation but sets a Storyline variable to true, dynamically revealing the Next button in the Storyline player.

Similar to the Rive app experiment, this test involved Lottie animations following a tutorial by Gary Simon. Embedded within a web object, the objective was to compare animation quality between Rive and Lottie. Clicking the 'GET STARTED' button triggers a lively confetti animation, showcasing the capabilities and visual appeal of Lottie in this interactive comparison.
